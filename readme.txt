# Editable CSS ReadMe

## Plugin Instructions

### All CSS styles added to the Editable CSS plugin will over write all styling previously present through the currently active WordPress theme.  Editing certain styles can cause undesired results, proceed with caution.

1.  Click on the "Appearance" section in the admin sidebar, then click the "CF CSS" link under the navigation
2.  The plugin displays a text area on the page for editing the CSS file
3.  To add or remove CSS from the file, add or remove CSS from the text area
4.  To save the changes to the CSS styling click the "Save CSS Changes" button at the bottom of the page
